# AI Workflow Assistant (Expo + Expo Router)

Local-first mobile concierge for fabrication drawings, assemblies, and packing data. Built with Expo, Expo Router, NativeWind, Zustand, and React Query.

## Getting started
1. Install deps: `npm install`
2. Run: `npx expo start`
3. Update the local AI endpoint in **Settings** (defaults to `http://localhost:11434`).
